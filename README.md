## ✅Rules

- 블라인드 오목은 각자의 말의 원래 색상을 구분할 수 없는 바둑돌을 사용하는 오목이다.
- 선공이 흑돌, 후공이 백돌이 된다.
- 바둑돌은 밑면을 제외한 면이 7가지 색상(빨강, 주황, 노랑, 초록, 파랑, 보라, 분홍)중에서 플레이어가 선택할 수 있다.
- 33, 44, 6목과 같은 일반적인 렌주룰의 금수가 허용된다
